41 

42 Parameters 

43 ---------- 

44 imgs : 3D/4D Niimg-like object 

45 Images to be masked. Can be 3-dimensional or 4-dimensional. 

46 

47 For all other parameters refer to NiftiMasker documentation. 

48 

49 Returns

51 signals : 2D numpy array 

52 Signals extracted using the provided mask. It is a scikit-learn

84class NiftiMasker(BaseMasker, CacheMixin): 

85 """Applying a mask to extract time-series from Niimg-like objects. 

86 

87 NiftiMasker is useful when preprocessing (detrending, standardization, 

88 resampling, etc.) of in-mask voxels is necessary. Use case: working with 

89 time series of resting-state or task maps. 

90 

91 Parameters 

92 ---------- 

93 mask_img : Niimg-like object, optional 

94 See http://nilearn.github.io/manipulating_images/input_output.html 

95 Mask for the data. If not given, a mask is computed in the fit step. 

96 Optional parameters (mask_args and mask_strategy) can be set to 

97 fine tune the mask extraction. If the mask and the images have different 

98 resolutions, the images are resampled to the mask resolution. If target_shape 

99 and/or target_affine are provided, the mask is resampled first. 

100 After this, the images are resampled to the resampled mask. 

101 

102 sessions : numpy array, optional 

103 Add a session level to the preprocessing. Each session will be 

104 detrended independently. Must be a 1D array of n_samples elements. 

105 

106 smoothing_fwhm : float, optional 

107 If smoothing_fwhm is not None, it gives the full-width half maximum in 

108 millimeters of the spatial smoothing to apply to the signal. 

109 

110 standardize : {False, True, 'zscore', 'psc'}, optional 

111 Strategy to standardize the signal. 

112 'zscore': the signal is z-scored. Timeseries are shifted 

113 to zero mean and scaled to unit variance. 

114 'psc': Timeseries are shifted to zero mean value and scaled 

115 to percent signal change (as compared to original mean signal). 

116 True : the signal is z-scored. Timeseries are shifted 

117 to zero mean and scaled to unit variance. 

118 False : Do not standardize the data. 

119 Default=False. 

120 

121 standardize_confounds : boolean, optional 

122 If standardize_confounds is True, the confounds are z-scored: 

123 their mean is put to 0 and their variance to 1 in the time dimension. 

124 Default=True. 

125 

126 high_variance_confounds : boolean, optional 

127 If True, high variance confounds are computed on provided image with 

128 :func:`nilearn.image.high_variance_confounds` and default parameters 

129 and regressed out. Default=False. 

130 

131 detrend : boolean, optional 

132 This parameter is passed to signal.clean. Please see the related 

133 documentation for details: :func:`nilearn.signal.clean`. 

134 Default=False. 

135 

136 low_pass : None or float, optional 

137 This parameter is passed to signal.clean. Please see the related 

138 documentation for details: :func:`nilearn.signal.clean`. 

139 

140 high_pass : None or float, optional 

141 This parameter is passed to signal.clean. Please see the related 

142 documentation for details: :func:`nilearn.signal.clean`. 

143 

144 t_r : float, optional 

145 This parameter is passed to signal.clean. Please see the related 

146 documentation for details: :func:`nilearn.signal.clean`. 

147 

148 target_affine : 3x3 or 4x4 matrix, optional 

149 This parameter is passed to image.resample_img. Please see the 

150 related documentation for details. 

151 

152 target_shape : 3-tuple of integers, optional 

153 This parameter is passed to image.resample_img. Please see the 

154 related documentation for details. 

155 

156 mask_strategy : {'background', 'epi' or 'template'}, optional 

157 The strategy used to compute the mask: use 'background' if your 

158 images present a clear homogeneous background, 'epi' if they 

159 are raw EPI images, or you could use 'template' which will 

160 extract the gray matter part of your data by resampling the MNI152 

161 brain mask for your data's field of view. 

162 Depending on this value, the mask will be computed from 

163 masking.compute_background_mask, masking.compute_epi_mask or 

164 masking.compute_brain_mask. Default='background'. 

165 

166 mask_args : dict, optional 

167 If mask is None, these are additional parameters passed to 

168 masking.compute_background_mask or masking.compute_epi_mask 

169 to fine-tune mask computation. Please see the related documentation 

170 for details. 

171 

172 sample_mask : Any type compatible with numpy-array indexing, optional 

173 Masks the niimgs along time/fourth dimension. This complements 

174 3D masking by the mask_img argument. This masking step is applied 

175 before data preprocessing at the beginning of NiftiMasker.transform. 

176 This is useful to perform data subselection as part of a scikit-learn 

177 pipeline. 

178 

179 dtype : {dtype, "auto"}, optional 

180 Data type toward which the data should be converted. If "auto", the 

181 data will be converted to int32 if dtype is discrete and float32 if it 

182 is continuous. 

183 

184 memory : instance of joblib.Memory or string, optional 

185 Used to cache the masking process. 

186 By default, no caching is done. If a string is given, it is the 

187 path to the caching directory. 

188 

189 memory_level : integer, optional 

190 Rough estimator of the amount of memory used by caching. Higher value 

191 means more memory for caching. Default=1. 

192 

193 verbose : integer, optional 

194 Indicate the level of verbosity. By default, nothing is printed. 

195 Default=0. 

196 

197 reports : boolean, optional 

198 If set to True, data is saved in order to produce a report. 

199 Default=True. 

200 

201 Attributes 

202 ---------- 

203 `mask_img_` : nibabel.Nifti1Image 

204 The mask of the data, or the computed one. 

205 

206 `affine_` : 4x4 numpy array 

207 Affine of the transformed image. 

208 

209 See also 

210 -------- 

211 nilearn.masking.compute_background_mask 

212 nilearn.masking.compute_epi_mask 

213 nilearn.image.resample_img 

214 nilearn.image.high_variance_confounds 

215 nilearn.masking.apply_mask 

216 nilearn.signal.clean 

217 

218 """

336 def fit(self, imgs=None, y=None): 

337 """Compute the mask corresponding to the data 

338 

339 Parameters 

340 ---------- 

341 imgs : list of Niimg-like objects 

342 See http://nilearn.github.io/manipulating_images/input_output.html 

343 Data on which the mask must be calculated. If this is a list, 

344 the affine is considered the same for all. 

345 

346 """ 

347 # y=None is for scikit-learn compatibility (unused here). 

348 

349 # Load data (if filenames are given, load them) 

350 if self.verbose > 0: 

351 print("[%s.fit] Loading data from %s" % ( 

352 self.__class__.__name__, 

353 _utils._repr_niimgs(imgs, shorten=False))) 

354 

355 # Compute the mask if not given by the user 

356 if self.mask_img is None: 

357 mask_args = (self.mask_args if self.mask_args is not None 

358 else {}) 

359 if self.mask_strategy == 'background': 

360 compute_mask = masking.compute_background_mask 

361 elif self.mask_strategy == 'epi': 

362 compute_mask = masking.compute_epi_mask 

363 elif self.mask_strategy == 'template': 

364 compute_mask = masking.compute_brain_mask 

365 else: 

366 raise ValueError("Unknown value of mask_strategy '%s'. " 

367 "Acceptable values are 'background', " 

368 "'epi' and 'template'." % self.mask_strategy) 

369 if self.verbose > 0: 

370 print("[%s.fit] Computing the mask" % self.__class__.__name__) 

371 self.mask_img_ = self._cache(compute_mask, ignore=['verbose'])( 

372 imgs, verbose=max(0, self.verbose - 1), **mask_args) 

373 else: 

374 self.mask_img_ = _utils.check_niimg_3d(self.mask_img) 

375 

376 if self.reports: # save inputs for reporting 

377 self._reporting_data = {'images': imgs, 'mask': self.mask_img_} 

378 else: 

379 self._reporting_data = None 

380 

381 # If resampling is requested, resample also the mask 

382 # Resampling: allows the user to change the affine, the shape or both 

383 if self.verbose > 0: 

384 print("[%s.fit] Resampling mask" % self.__class__.__name__) 

385 self.mask_img_ = self._cache(image.resample_img)( 

386 self.mask_img_, 

387 target_affine=self.target_affine, 

388 target_shape=self.target_shape, 

389 copy=False, interpolation='nearest') 

390 if self.target_affine is not None: # resample image to target affine 

391 self.affine_ = self.target_affine 

392 else: # resample image to mask affine 

393 self.affine_ = self.mask_img_.affine 

394 # Load data in memory 

395 get_data(self.mask_img_) 

396 if self.verbose > 10: 

397 print("[%s.fit] Finished fit" % self.__class__.__name__) 

398 

399 if (self.target_shape is not None) or (self.target_affine is not None): 

400 if self.reports: 

401 if imgs is not None: 

402 resampl_imgs = self._cache(image.resample_img)( 

403 imgs, target_affine=self.affine_, 

404 copy=False, interpolation='nearest') 

405 else: # imgs not provided to fit 

406 resampl_imgs = None 

407 self._reporting_data['transform'] = [resampl_imgs, self.mask_img_] 

408 

409 return self

411 def transform_single_imgs(self, imgs, confounds=None, copy=True): 

412 """Apply mask, spatial and temporal preprocessing 

413 

414 Parameters 

415 ---------- 

416 imgs : 3D/4D Niimg-like object 

417 See http://nilearn.github.io/manipulating_images/input_output.html 

418 Images to process. It must boil down to a 4D image with scans 

419 number as last dimension. 

420 

421 confounds : CSV file or array-like or pandas DataFrame, optional 

422 This parameter is passed to signal.clean. Please see the related 

423 documentation for details: :func:`nilearn.signal.clean`. 

424 shape: (number of scans, number of confounds) 

425 

426 copy : Boolean, optional 

427 Indicates whether a copy is returned or not. Default=True. 

428 

429 Returns 

430 ------- 

431 region_signals : 2D numpy.ndarray 

432 Signal for each voxel inside the mask. 

433 shape: (number of scans, number of voxels) 

434 

435 """ 

436 

437 # Ignore the mask-computing params: they are not useful and will 

438 # just invalid the cache for no good reason 

439 # target_shape and target_affine are conveyed implicitly in mask_img 

440 params = get_params(self.__class__, self, 

441 ignore=['mask_img', 'mask_args', 'mask_strategy']) 

442 

443 data = self._cache(filter_and_mask, 

444 ignore=['verbose', 'memory', 'memory_level', 

445 'copy'], 

446 shelve=self._shelving)( 

447 imgs, self.mask_img_, params, 

448 memory_level=self.memory_level, 

449 memory=self.memory, 

450 verbose=self.verbose, 

451 confounds=confounds, 

452 copy=copy, 

453 dtype=self.dtype 

454 ) 

455 

456 return data
