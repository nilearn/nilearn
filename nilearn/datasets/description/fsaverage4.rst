fsaverage4 template
===================

Access
------
See :func:`nilearn.datasets.fetch_surf_fsaverage`.

.. include:: fsaverage_content.rst

Notes
-----
Fsaverage4 standard surface as distributed with Freesurfer (:footcite:t:`Fischl1999`).

The mesh has 2562 vertices per hemisphere.

The order of vertices was modified to match that of fsaverage5-7.
