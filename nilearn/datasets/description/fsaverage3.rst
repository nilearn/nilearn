fsaverage3 template
===================

Access
------

Notes
-----
Fsaverage3 standard surface as distributed with Freesurfer (:footcite:t:`Fischl1999`).

The mesh has 642 vertices per hemisphere.

The order of vertices was modified to match that of fsaverage5-7.
